Google Sheet 설정

서비스 계정 JSON 키와 Sheet ID를 한 번 입력하면 브라우저에 저장됩니다 (이 기기에만).

Google Sheet 준비 방법

1. 본인 구글 계정으로 새 Google Sheet 생성
2. Google Cloud Console에서 새 프로젝트 → Sheets API 활성화
3. IAM & Admin → Service Accounts → 새 계정 생성 → Keys → Add Key → JSON 다운로드
4. JSON 파일의 client_email 값을 복사 후, 생성한 Sheet를 그 이메일로 공유 (편집자 권한)
5. Sheet URL에서 Sheet ID (/d/ 뒤의 긴 문자열) 복사
6. 위 양식에 JSON 전체와 Sheet ID 입력 → 저장 → 연결 테스트 → Sheet 초기화 (탭 생성)
